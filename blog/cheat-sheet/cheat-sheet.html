Cheat Sheet

DISCLAIMER: This is a Minimum Viable Project. I will be adding Ruby and JavaScript terms as well.

CSS

  • Opacity: makes an element transparent using an

    alpha
    value. It's a number between 0 and 1.0 (for example, 0.5 is 50% transparency).

    element {

    background-color: #000000;

    opacity: 0.5;

    }

  • Font Weight: allows you to make text either normal or bold.

    element {

    font-weight: bold;

    }

  • Text transform: changes text to all upper case, lower case, or capitalizes the first letter of every word.

    element1 {

    text-transform: uppercase;

    }

    element2 {

    text-transform: lowercase;

    }

    element3 {

    text-transform: capitalize;

    }

  • Text decoration: allows you to remove all lines from text, add an underline, add a line above text, put a line through text, or make text blink. DO NOT use blinking text, it's really annoying and it's bad practice to use it. Seriously. I'm not even going to put it in the example.

    element1 {

    text-decoration: none;

    }

    element2 {

    text-decoration: underline;

    }

    element3 {

    text-decoration: overline;

    }

    element4 {

    text-decoration: line-through;

    }

  • Text indent: indents text using a pixel or em specification.

    element {

    text-indent: 20px;

    }

  • Text shadow: drops a shadow behind text using 4 specification fields: left to right shadow, top to bottom shadow, shadow blur, and shadow color. The first three are measured in pixels or ems, and the fourth is a hex or rgb color.

    element {

    text-shadow: 1em 2em 1em #000000;

    }

  • Box shadow: drops a shadow behind a box using 5 specification fields: horizontal(left to right) offset, vertical(top to bottom) offset, blur distance(blurs the edges of the shadow), shadow spread(positive expand the shadow, negative values shrink it), and shadow color. The first 4 fields are measured in pixels and ems, the last field is a hex or rgb color value. One important thing to note-the first four fields can also be preceeded by the word "inset," which will drop the shadow inside of the box instead of outside.

    element {

    text-shadow: 3em 3em 1em 1em #ffffff;

    }

  • Border radius: rounds the corners of a box. You can just use

    border-radius
    with one pixel or em value to round all four corners with the same style at once. You can also use four pixel or em values to round the top right, bottom right, bottom left and top left corners with different values. Lastly, you can use
    border-top-right-radius, border-bottom-right-radius, border-bottom-left-radius,
    and
    border-top-left-radius
    to style each respective corner.

    element1 {

    border-radius: 1em 1em 1em 1em;

    }

    element2 {

    border-top-right-radius: 1em;

    border-bottom-right-radius: 1em;

    border-bottom-left-radius: 1em;

    border-top-left-radius: 1em;

    }
