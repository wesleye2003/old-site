CSS Concepts

Relative, Absolute, Fixed, and Static Positioning

2/7/2016

Static positioning
is the default position of all elements; when in static position, all elements appear in the normal flow (on top of each other in the order the web browser sees them in the code).

Absolute positioning
removes an element from the normal flow and moves an element toward the top, right, left, or bottom by using its default position in the normal flow as a reference point. Basically, it takes an element out of the normal flow and moves it around without affecting other elements.

Relative positioning
moves an element from where it was in the normal flow while still holding that element's place in the normal flow. In a sense, it is the opposite of absolute positioning; relative positioning moves an element in the normal flow *while* keeping its original space in the normal flow intact.

Fixed positioning
takes an element out of the normal flow (like absolute positioning ) and places it in a fixed location in the browser window. Just like with absolute positioning, an element that is placed in fixed position will no longer affect elements that follow the normal flow of the code.
