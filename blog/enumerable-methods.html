Ruby: Enumerable Methods

The .count Method

2/21/16

Enumerables
are a type of classification for many objects in Ruby, and by calling and Enumerable method you can modify data in one of the qualifying sub-classes of Enumerable. I'll be talking about an Enumerable method called
#count
. When called, the
#count
method takes an Enumerable as its input and returns the number of items in that Enumerable if no value or block of code is supplied. If a value is also specified,
#count
will return the number of items equal to that value; the same concept applies to a block of code. The method will return the number of items equal to the return value of whatever code is supplied. Here's an example(the code is on the left, the returned value is to the right of the #=>):

array = [1, 2, 2, 4, 4, 6, 7]

array.count #=> 7

array.count(4) #=> 2

array.count{ |n| n%2==0 } #=> 5

The last example may be a bit hard to read but the code is simple.
#count
iterates through the array and adds 1 to the total returned value everytime the remainder of the term in the array divided by 2 is equal to zero. Basically, that block of code checks for numbers that are divisible by 2.
